# 🏥 Healthcare Analytics Dashboard | Power BI

## 📌 Project Overview

This project is an interactive **Healthcare Analytics Dashboard** built using **Microsoft Power BI**. It provides valuable insights into hospital operations, patient records, billing, diagnoses, insurance coverage, doctor performance, and bed occupancy.

The dashboard enables healthcare administrators and decision-makers to monitor key performance indicators (KPIs), identify trends, and make data-driven decisions.

---

## 🎯 Objectives

- Analyze patient admission and discharge records
- Monitor hospital billing and insurance amounts
- Track diagnosis distribution
- Visualize doctor performance and patient feedback
- Analyze bed occupancy trends
- Provide interactive filtering for quick insights

---

## 📊 Dashboard Features

### Key KPIs
- Total Billing Amount
- Earliest Admission Date
- Earliest Discharge Date
- Earliest Follow-up Date

### Interactive Filters
- Patient ID
- Date Range

### Visualizations
- 📈 Billing Amount Trend
- 🛏️ Bed Occupancy Distribution
- 👨‍⚕️ Doctor-wise Feedback
- 🩺 Diagnosis Funnel Analysis
- 💰 Health Insurance vs Billing Analysis

- How are hospital beds occupied?
